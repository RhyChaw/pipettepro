# Quiz Meme Images

## Required Images

You need to add **3 encouraging meme images** to this folder for the quiz results modal.

### File Names:
- `meme-1.jpg` (or `.png`)
- `meme-2.jpg` (or `.png`)
- `meme-3.jpg` (or `.png`)

### Image Requirements:
- **Format**: JPG or PNG
- **Recommended Size**: 320x240 pixels (or similar aspect ratio)
- **Content**: Encouraging, motivational images related to learning, science, or success
- **File Size**: Keep under 500KB each for fast loading

### Suggested Content:
1. **meme-1.jpg**: Image with text like "Great job! Even when experiments don't go as planned, every attempt is a step forward."
2. **meme-2.jpg**: Image with text like "You're doing amazing! Science is all about persistence."
3. **meme-3.jpg**: Image with text like "Fantastic work! Remember, every expert was once a beginner."

### Where to Find Images:
- Create your own using meme generators
- Use royalty-free images from Unsplash, Pexels, or Pixabay
- Design custom graphics with encouraging messages
- Use science/lab-themed motivational images
